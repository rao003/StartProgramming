## Basic Commands

You are now in a game of Minecraft! Go walk around, hack things, and build things!

Use the mouse to look around and use the following keys on the keyboard:

Key	Action
W	Forward
A	Left
S	Backward
D	Right
E	Inventory
Space	Jump
Double Space	Fly / Fall
Esc	Pause / Game menu
Tab	Release mouse cursor
